# React-jack

### A single-player blackjack game built using React and Redux

To get up and running, clone the repo, cd into it and type
```
$ npm install
```
or
```
$ yarn install
```


To launch the app, type
```
$ npm run start
```
or 
```
$ yarn start
```


To run the test suite, type
```
$ npm run test
````
or
```
$ yarn test
```
